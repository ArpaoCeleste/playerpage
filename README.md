## About

Official portfolio website for Mariana Queirós, professional football player at FC Porto Feminino. The website showcases her career trajectory, achievements, media gallery, and professional highlights.

## Features

- Modern responsive design with particle animations
- Professional career timeline
- Trophy and achievements showcase
- Media gallery with photos and videos
- National team representations overview
- Smooth scroll animations and transitions
- Mobile-first responsive layout
- SEO optimized with structured data
- Google Analytics integration

## Technologies

### Frontend
- **PHP** - Server-side rendering
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript** - Interactive features
- **Particles.js** - Particle animations
- **Font Awesome** - Icon library

### Features
- Responsive design
- Smooth scroll animations
- Dynamic typing effect
- Image lazy loading
- Optimized performance
- SEO optimization with Schema.org markup

### SEO & Analytics
- Google Analytics (G-Q4QHJGKRQJ)
- Open Graph meta tags
- Twitter Cards
- Schema.org structured data
- Canonical URLs

## Pages

### Homepage (index.php)
- Hero section with player introduction
- Career trajectory timeline
- National team representations
- Trophy showcase (Palmarés)

### About (sobre.php)
- Detailed biography
- Professional information

### Media (media.php)
- Photo gallery
- Video highlights
- Press coverage
- Social media integration

## Installation

1. Clone the repository
```bash
git clone https://github.com/ArpaoCeleste/playerpage.git
cd playerpage
```

2. Configure web server (Apache/Nginx)
Ensure PHP is installed and configured

3. Update the base URL
Edit header.php to match your domain:
```php
$baseUrl = "https://yourdomain.com";
```

4. Deploy to web server
Upload files to your hosting provider

## Requirements

- PHP 7.4 or higher
- Web server (Apache/Nginx)
- SSL certificate (recommended for HTTPS)
